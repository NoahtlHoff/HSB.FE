
# HackStreeBoys_Website

This is a web application built with ASP.NET Core. It features user authentication (registration and login) and a real-time chat application.

# Link to project

https://hackstreeboyswebsite20251023145048-dfagc5fzaqcufqfm.francecentral-01.azurewebsites.net/

## Features

*   **User Authentication:** Users can register for a new account and log in with existing credentials.
*   **Chat:** Logged-in users can access a chat page to send and receive messages in real-time.
*   **Frontend:** The frontend is built using Razor Views and Bootstrap for styling.
